Secunda

Posted 8 days ago

I published this theme a while ago, but didn’t post a devlog until now.

This is my current theme for Visual Studio Code which a fork of the Ra Eyeful theme. Ra Eyeful already fits well with my current system’s theme but I made more tweaks to fix minor inconsistencies.

I added a bunch of rules for VS Code’s user interface elements, such as buttons, form elements, etc. and I also added colors for the integrated terminal.

To install it, launch VS Code’s Quick Open (Ctrl+P) interface and enter this command:

ext install ruj.secunda

For more information about the extension, you can view Secunda on the Visual Studio Marketplace or at Secunda’s GitHub repository.
