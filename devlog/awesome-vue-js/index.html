Awesome Vue.js

Posted 25 days ago (updated 6 minutes ago)

My fork of the Vue.js awesome list now has 100+ stargazers. It’s quite surprising to me that it reached a lot of people considering I haven’t really promoted it anywhere (before this post, that is!).

Awesome Vue.js is, I guess, what they would call an “awesome list”. An awesome list is a collection of links to interesting projects or libraries realated to a particular topic. The topic, in this case, is Vue.js which is a JavaScript framework for building web sites and web applications.

These lists are typically coded into a single text file and are available to everybody through a public domain license or something similar.

Vue.js is very popular to software developers so it’s not surprising that people use it on a lot of web applications and libraries. To keep track of these projects, the Vue.js team have their own awesome list repository which I forked.

My fork is essentially a copy of the official awesome list. I pull the changes from upstream and make some tiny fixes, if necessary. The fork also has a web site which aims to provide a nicer experience when scanning the list.

So, if you haven’t already, please check out the website. You can also view the source on GitHub.
